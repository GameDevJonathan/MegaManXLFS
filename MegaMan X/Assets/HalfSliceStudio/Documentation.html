Easy Scene Loader Documentation

Hey there! Welcome to the documentation for the Easy Scene Loader extension. This handy tool makes loading and managing scenes in Unity a breeze. Let's dive in!

Installation

  1. Grab the Easy Scene Loader extension package.
  2. Import it into your Unity project.
  3. You're all set! The extension will be ready to use.

Opening the Easy Scene Loader Window

To open the Easy Scene Loader window, simply go to Tools > Easy Scene Loader in the Unity menu bar. The window will pop up, and you'll see all your project's scenes organized into categories.

Loading Scenes

Loading scenes is a piece of cake with Easy Scene Loader:

  • To load a scene in standard mode, just click on the scene name button. This will replace the currently active scene.
  • If you want to load a scene additively (keeping the current scene loaded), hit the "Additive" button next to the scene name. The new scene will be loaded alongside the active scene.

Unloading Scenes

Need to unload a scene that was loaded additively? No problem! If a scene was loaded additively, you'll see an "Unload" button next to it. Just click that button, and the scene will be unloaded, freeing up resources.

Preloading Scenes

Easy Scene Loader also lets you preload all the scenes in your project to reduce loading times during gameplay. Here's how:

  1. Expand the "Preload Settings" section in the Easy Scene Loader window.
  2. Set the desired preload delay (in seconds) using the "Preload Delay" field. This delay gives you time to set up any necessary game state before the scenes are preloaded.
  3. Click the "Preload All Scenes" button to start preloading all the scenes additively in play mode.

After the preload delay, all the scenes will be loaded in the background, ready to be accessed quickly when needed.

Unloading Unused Scenes

To free up memory, Easy Scene Loader can automatically unload scenes that were preloaded but are no longer in use. Here's how it works:

  1. Expand the "Unload Settings" section in the Easy Scene Loader window.
  2. Set the desired unload delay (in seconds) using the "Unload Delay" field. This delay determines how long preloaded scenes will stay in memory before being unloaded if unused.
  3. Click the "Unload Unused Scenes" button to initiate the delayed unloading process.

After the unload delay, any preloaded scenes that are not currently in use will be automatically unloaded, freeing up memory.

Scene Categories

Easy Scene Loader organizes your scenes into categories based on their folder structure. Scenes located in the same folder will be grouped together under a category name derived from the folder name. This makes it easy to find and manage related scenes.

Search Functionality

If you have a ton of scenes in your project, finding the one you need can be a hassle. That's where the search functionality comes in handy. Just type in the name or partial name of the scene you're looking for in the search bar, and Easy Scene Loader will filter the scene list to show only matching scenes. It's like having a search engine for your scenes!

Refresh Scene List

If you've added or removed scenes from your project, you can easily refresh the scene list in Easy Scene Loader by clicking the "Refresh Scene List" button. This will update the list to reflect the current state of your project's scenes.

Wrapping Up

That's it, folks! You're now equipped with the knowledge to use Easy Scene Loader like a pro. Go ahead and streamline your scene management workflow, and happy coding!
